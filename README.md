# 🤖 AI Resume Reviewer

An AI-powered Resume Reviewer built with **Python** and the **Google Gemini API** that analyzes resumes against job descriptions and provides intelligent feedback to improve resume quality and ATS compatibility.

---

## 📌 Features

- 📄 Analyze resumes using AI
- 🎯 Compare resumes with job descriptions
- 💡 Generate intelligent improvement suggestions
- ✅ Improve ATS compatibility
- 🔒 Secure API key management using environment variables (.env)
- 🧩 Modular and maintainable project structure

---

## 🛠️ Tech Stack

- Python
- Google Gemini API
- REST APIs
- Git & GitHub
- VS Code

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/Snehardh/resume-reviewer.git
```

Go to the project folder:

```bash
cd resume-reviewer
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Create a `.env` file and add your Gemini API Key.

Run the application:

```bash
python main.py
```
