# Resume Reviewer AI

An AI-powered Resume Reviewer that analyzes a resume against a job description and provides detailed feedback using Google Gemini AI.

## Features

- Analyzes resume against a job description
- Provides match score out of 10
- Identifies key strengths
- Highlights missing skills
- Gives improvement suggestions

## Tech Stack

- Python 3.8+
- Google Gemini AI API
- Requests
- Python-dotenv

## Setup

1. Clone the repo
2. Create virtual environment
3. Install dependencies: pip install requests python-dotenv
4. Add your API key to .env file
5. Run: python app.py
